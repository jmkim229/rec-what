🎮 게임 방법

1

모든 참가자는 "슬라임"으로 시작한다. 슬라임 자세(쪼그려 앉아 펭귄걸음, "뿅뿅" 소리)로 돌아다닌다.

2

같은 등급의 다른 사람을 찾아 가위바위보를 한다. 다른 등급과는 가위바위보 불가. 이긴 사람은 한 단계 진화, 진 사람은 한 단계 강등된다. (단, 슬라임은 더 강등되지 않음)

3

진화하거나 강등될 때마다 해당 등급의 제스처로 즉시 바꿔 돌아다니며 같은 등급의 상대를 찾는다. 승부 결과를 "진화!" / "강등!"이라고 크게 외치면 분위기가 산다.

4

드래곤에서 가위바위보에 이기면 왕이 된다! 왕은 왕좌(미리 정해둔 의자)에 앉아 도전자를 기다린다. 왕은 항상 1명만 유지된다.

5

진행자가 정한 제한시간(보통 10분)이 끝날 때 왕좌에 앉아있는 사람이 최종 우승!

👑 왕 도전 규칙

⚡ 드래곤만 왕에게 도전할 수 있다.
🐉 드래곤 승리 → 기존 왕이 드래곤으로 강등, 도전자가 새 왕이 된다.
👑 왕 승리 → 도전한 드래곤이 오크로 강등된다.
핵심 규칙 요약
• 같은 등급끼리만 가위바위보 가능
• 이기면 한 단계 진화 → 진 사람은 한 단계 강등
• 슬라임은 강등 없음 | 왕은 1명만 존재
• 드래곤만 왕에게 도전 가능

💡 진행 팁

제스처 사전 연습이 핵심!
각 등급 제스처는 게임 시작 전 모두 함께 연습해보세요. 멀리서도 등급이 보여야 재밌어요.
  • 진행자는 종이나 호루라기로 게임 시작·종료를 명확히 알려주세요.
  • 신나는 배경음악이 있으면 분위기가 두 배가 됩니다.
  • 인원이 30명 이상일 때는 단계를 6~7단계로 늘려도 좋아요. (예: 슬라임 → 고블린 → 오크 → 트롤 → 드래곤 → 데몬 → 왕)
  • 가위바위보 후 승부 외칠 때 "진화!" "강등!"을 큰 소리로 외치게 하면 분위기가 살아납니다.
  • 왕좌 의자가 없다면 바닥에 원을 그려 "왕의 자리"로 표시해도 됩니다.
  • 사람이 많을 때는 부딪힘 주의! 충분히 넓은 공간에서 진행하세요.

🔄 변형 방법

⚔️ 팀 왕좌전 두 팀으로 나눠서 진행. 같은 팀끼리는 가위바위보 불가, 상대 팀과만 승부. 시간 종료 시 양 팀의 진화 점수 합계로 승부를 가린다.
등급 점수
🟢 슬라임1점
👹 고블린2점
🧌 오크3점
🐉 드래곤4점
👑 왕10점
🌀 무한 강등 모드 강등 시 한 단계가 아닌 무조건 슬라임으로 돌아가는 하드코어 버전. 긴장감이 극대화되고 끝까지 변수가 발생한다.
🃏 카드 진화 버전 가위바위보 대신 트럼프 카드를 사용. 각자 카드 한 장씩 들고 상대와 비교해 높은 숫자가 승리. 무승부도 가능해서 더 짜릿하다.
⚡ 묵찌빠 진화 가위바위보 대신 "묵찌빠"로 진행. 한 번에 승부가 나지 않아 더 박진감 넘친다.
