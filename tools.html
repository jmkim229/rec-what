레크리에이션 진행자 도구는 게임 진행에 필요한 타이머·룰렛·역할 배정·출제기를 화면에서 바로 사용할 수 있는 도구입니다. 스마트폰이나 노트북 하나면 충분합니다. 앱 설치 없이, 준비물 없이 바로 진행해보세요.

⏱️ 타이머 & 랜덤 도구

💣

폭탄 돌리기 타이머

랜덤 타이머와 스톱워치로 게임 타이밍을 자유롭게 제어. 언제 폭탄이 터질지 아무도 모른다!

타이머 ⏱️

방탈출 타이머

제한시간 설정, 힌트 카운터, 긴장감 넘치는 경고 효과 포함. 방탈출 게임을 더 몰입감 있게 진행.

타이머 ⏱️

찜피구 도구

팀 자동 나누기, 라운드 타이머, 공수 교대 기능. 찜피구 진행을 훨씬 편하게.

타이머 🏃

릴레이 미션 타이머

팀 릴레이 미션의 진행 시간을 측정하고 라운드를 관리하는 진행자 전용 타이머.

타이머

🎲 역할 배정 & 진행 도구

🕵️

마피아 진행 도구

밤/낮 페이즈 자동 진행, 역할 호출, 효과음, 토론 타이머. 진행자가 폰 하나로 모두 진행 가능.

역할 배정 🕵️

라이어 게임 진행 도구

참가자 수와 카테고리를 설정하면 라이어를 자동으로 배정. 카테고리별 키워드도 제공.

역할 배정 🔢

업다운 진행 도구

비밀 숫자 설정과 업·다운 힌트로 업다운 게임을 손쉽게 진행. 범위와 난이도 조절 가능.

진행 도구 ⚡

스피드 퀴즈 진행 도구

타이머와 점수판으로 팀 대항 스피드 퀴즈를 손쉽게 진행. 문제 목록 관리도 간편.

진행 도구 🔢

스무고개 질문 카운터

남은 질문 수를 큰 화면에 표시하고 예·아니오 답변을 자동 기록. 질문 수 10~30개 설정, 정답 선언 판정까지.

진행 도구

🃏 출제기 & 뽑기 도구

🔤

초성퀴즈 자동 출제기

단어를 입력하면 초성을 자동 추출해 큰 화면에 표시. 카테고리 선택, 단계별 힌트, 타이머 포함.

출제기 ⚖️

밸런스게임 질문 뽑기

카테고리별 랜덤 질문 자동 출제, 즉석 득표 집계, 직접 질문 추가 기능. 발표 없이 화면만 보여주면 끝.

뽑기 🙌

몸으로 말해요 제시어 뽑기

쉬움·보통·어려움 난이도별, 주제별 제시어를 즉시 뽑아줍니다. 이제 제시어 찾느라 시간 낭비 없이!

뽑기 🙋

너도나도 주제 추천 도구

카테고리를 선택하면 너도나도 게임 주제를 랜덤으로 뽑아줍니다. 매번 새로운 주제로 진행.

뽑기 🎤

이구동성 단어 출제기

팀원 수(글자 수)를 정하면 그 인원에 맞는 단어만 랜덤 출제. 글자 수가 딱 맞아야 성공!

출제기 🐘

동물 카드 뽑기 도구

동물 흉내 게임 진행자 도구. 동물 카드를 한 장씩 뽑아 화면에 크게 표시합니다.

뽑기 🏐

풍선 배구 점수판

코트를 탭하면 득점! 세트 스코어·서브 팀·듀스까지 폰 하나로 관리하는 점수판입니다.

진행 도구 🎨

색깔 찾기 도구

난이도·자동/수동 모드를 설정하면 화면 전체를 해당 색으로 채워 줍니다. 색깔 찾기 게임을 더 쉽게.

진행 도구 🃏

카드 뒤집기 도구

학습용 카드 세트, 팀별 점수, 미리보기 기능 포함. 기억력 짝맞추기 게임을 화면으로 진행.

레크 진행 도구를 이렇게 활용하세요

  • 진행자가 스마트폰을 빔프로젝터·TV에 연결하면 전체 화면 공유 가능
  • 도구 페이지는 인터넷 없이도 동작하는 경우가 많아 현장에서 바로 사용 가능
  • 각 도구 페이지에서 게임 설명 페이지로 바로 이동할 수 있습니다
